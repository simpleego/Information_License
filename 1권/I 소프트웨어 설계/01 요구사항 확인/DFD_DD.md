### 자료 흐름도 작성 지침
- 처리 후 변환되면 새로운 이름 부여  
- 출력 위해 입력 필요  
- 상위·하위 자료 흐름 일치  
- 입력 있다고 출력 반드시 필요한 것은 아님

## 자료사전 작성 예제
① 고객 주문 정보
* 주문정보 = 주문번호 + 주문일자 + 고객명 + {주문항목} + (배송요청사항)
* 주문항목 = 상품명 + 수량 + 단가
* 배송요청사항 = *부재 시 경비실에 맡겨주세요 등*
코드를 사용할 때는 주의가 필요합니다.

② 회원 가입 정보
* 회원등록 = 아이디 + 비밀번호 + 이름 + 연락처 + (이메일)
* 연락처 = [ 휴대폰번호 | 유선전화번호 ]
* 성별 = [ "남" | "여" ]

③ 학생 성적 처리
* 성적표 = 학번 + 이름 + {과목성적}1^8 + 총점 + 평균
* 과목성적 = 과목코드 + 과목명 + 점수
* 주석 = *과목성적은 최소 1개에서 최대 8개까지 가능*

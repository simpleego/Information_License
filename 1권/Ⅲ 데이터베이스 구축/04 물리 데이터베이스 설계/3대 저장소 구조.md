## Storage Types (DAS, NAS & SAN)
```markdown
| Category | DAS | NAS | SAN |
|----------|-----|-----|-----|
| **Protocols** | SATA, SAS | SMB (CIFS), NFS | iSCSI, Fiber Channel |
| **Type** | File Storage | Block Storage | Block Storage |
| **Speed** | 5–10 ms | 5–10 ms | 20–50 ms |
| **Data Transmission** | IDE / SCSI | Ethernet, TCP/IP | Fiber Channel |
| **Complexity** | Easy | Moderate | Complex |
| **Management Costs** | High | Moderate | Low |
| **Storage Type** | Sectors | Shared Files | Blocks |
| **Supports Capacity Sharing?** | No (possible manually) | Yes | Yes |
| **Connected to Network?** | No | Yes | Yes |
| **Scalability** | Low | Medium | High |
```

원하시면 **한글 버전 테이블**로도 변환해 드릴 수 있습니다.
예: 항목명을 `분류 / 프로토콜 / 속도 / 확장성` 등으로 번역한 표.

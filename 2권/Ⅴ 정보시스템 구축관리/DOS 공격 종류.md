#  DOS 공격종류

주요 DoS 공격 유형
## TCP SYN Flooding: TCP 연결 설정 시(3-way handshaking) 반만 연결된 상태(Half-open)를 다수 생성하여 서버의 접속 테이블 자원을 고갈시키는 방식.  

## UDP/ICMP/SYN Flooding (플러딩): 대량의 패킷을 한 번에 전송하여 네트워크 대역폭이나 시스템 자원을 포화 상태로 만듦.

## Smurf (스머프) 공격: IP 주소를 공격 대상의 것으로 위조하여 ICMP Echo Request(핑)를 브로드캐스트로 전송, 응답 트래픽이 집중되도록 유도.

## TearDrop (티어드롭) 공격: 패킷이 분할(Fragmentation)되어 전송될 때 오프셋 값을 조작하여 수신 시스템이 재조립 시 오류를 일으켜 시스템을 다운시킴.

## Ping of Death (PoD, 핑 오브 데스): 정상적인 핑(Ping) 패킷보다 훨씬 큰 크기로 데이터를 보내 IP 프래그먼트 재조립 시 메모리 버퍼 오버플로우를 유발.

## Land Attack (랜드 어택): 공격 패킷의 출발지 IP와 포트, 목적지 IP와 포트를 동일하게 설정하여 대상 시스템이 자기 자신에게 응답을 계속 보내게 하여 마비시킴.

## 애플리케이션 계층 공격 (Slowloris, HTTP GET Flood): 정상적인 요청처럼 보이지만, 아주 느리게 연결을 유지하거나 지속적인 GET 요청을 보내 웹 서버의 연결 제한을 소진시킴

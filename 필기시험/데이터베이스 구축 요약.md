# **정보처리기사 데이터베이스 구축 요약

---

## 📌 핵심 학습 영역 (4시간 전략)

### 1. 데이터 모델링 & 정규화 (1시간)
- **개념적/논리적/물리적 데이터 모델링**  
- **ERD(Entity Relationship Diagram)**: 엔티티, 속성, 관계  
- **정규화 단계**: 제1정규형(원자값), 제2정규형(부분 함수 종속 제거), 제3정규형(이행 종속 제거), BCNF  
- **반정규화**: 성능 향상을 위한 테이블 병합, 중복 허용  

👉 **정규화 단계별 특징과 ERD 표기법**은 단골 출제 포인트.

---

### 2. SQL (1시간 30분)
- **DDL**: CREATE, ALTER, DROP  
- **DML**: SELECT, INSERT, UPDATE, DELETE  
- **JOIN**: INNER, OUTER, SELF JOIN  
- **집계 함수**: COUNT, SUM, AVG, MAX, MIN  
- **서브쿼리 & 뷰(View)**  
- **인덱스, 트리거, 프로시저**  

👉 **JOIN과 서브쿼리 문제**가 자주 나오므로 반드시 연습하세요.

---

### 3. 트랜잭션 & 동시성 제어 (30분)
- **트랜잭션 특성(ACID)**: 원자성, 일관성, 독립성, 지속성  
- **동시성 제어 기법**: 로킹(Locking), 타임스탬프, 다중버전(MVCC)  
- **회복 기법**: 로그 기반 회복, 체크포인트  

👉 **ACID와 동시성 제어 기법 비교 문제**는 필수 암기.

---

### 4. DBMS 구조 & 기타 (1시간)
- **DBMS 아키텍처**: 3단계 구조(외부, 개념, 내부)  
- **스키마 종류**: 외부 스키마, 개념 스키마, 내부 스키마  
- **데이터 무결성 제약조건**: 개체, 참조, 도메인 무결성  
- **분산 DB, 병행 제어, 클러스터링**  

👉 **스키마와 무결성 제약조건**은 단답형으로 자주 출제됩니다.

---

## 📝 4시간 학습 로드맵
| 시간 | 학습 내용 | 목표 |
|------|-----------|------|
| 1시간 | 데이터 모델링·정규화 | ERD와 정규화 단계 암기 |
| 1시간 30분 | SQL | JOIN·서브쿼리·집계 함수 숙지 |
| 30분 | 트랜잭션·동시성 제어 | ACID와 제어 기법 정리 |
| 1시간 | DBMS 구조·무결성 | 3단계 구조와 제약조건 암기 |

---

## ⚡ 시험 대비 팁
- **SQL 문제는 직접 손으로 작성해보는 연습**이 가장 효과적입니다.  
- **정규화 단계와 무결성 제약조건**은 암기형 문제 대비 필수.  
- **기출문제 병행 학습**으로 출제 패턴을 빠르게 파악하세요.  

---

✅ 결론: **정규화·SQL·트랜잭션·DBMS 구조**를 4시간 안에 압축 학습하면 시험 대비에 가장 효율적입니다.

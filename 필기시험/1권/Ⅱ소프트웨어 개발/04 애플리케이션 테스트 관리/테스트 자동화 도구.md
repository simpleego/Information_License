# 자동화 도구
> 테스트 자동화 도구는 목적과 테스트 방식에 따라 여러 종류
> **종류별로 대표 제품까지 정리**

---

# 🧪 **1. 기능(UI) 테스트 자동화 도구**
사용자 인터페이스를 실제로 조작하는 방식의 자동화 도구들.

### 대표 제품
- **Selenium**  
- **Cypress**  
- **Playwright**  
- **Puppeteer**  
- **Appium** (모바일 앱)  
- **TestComplete**  
- **Ranorex**  
- **Katalon Studio**

### 특징
- 웹/모바일 UI 테스트 자동화  
- 크로스 브라우저 테스트 지원  
- 회귀 테스트에 많이 사용됨

---

# ⚙️ **2. API / 백엔드 테스트 자동화 도구**
UI 없이 API 레벨에서 기능을 검증하는 도구.

### 대표 제품
- **Postman**  
- **Rest Assured**  
- **SoapUI / ReadyAPI**  
- **Karate**  
- **JMeter** (부하 + API 테스트 가능)

### 특징
- REST, SOAP API 테스트  
- 자동화 스크립트 작성 용이  
- CI/CD 파이프라인에 쉽게 통합됨

---

# 🚀 **3. 성능 / 부하 테스트 도구**
시스템의 처리량, 응답 시간, 안정성을 측정하는 도구.

### 대표 제품
- **JMeter**  
- **Gatling**  
- **Locust**  
- **k6**  
- **LoadRunner**

### 특징
- 대규모 트래픽 시뮬레이션  
- TPS, 응답 시간, 에러율 측정  
- 서버 병목 구간 분석

---

# 🔍 **4. 단위 테스트(Unit Test) 프레임워크**
개발자가 코드 단위로 테스트를 자동화하는 도구.

### 대표 제품
- **JUnit** (Java)  
- **pytest / unittest** (Python)  
- **Jest / Mocha** (JavaScript)  
- **NUnit / xUnit** (.NET)  
- **Go testing** (Go)

### 특징
- 코드 레벨 테스트  
- TDD/BDD에 활용  
- CI에서 가장 많이 실행되는 테스트

---

# 🧩 **5. BDD(Behavior Driven Development) 도구**
시나리오 기반 테스트 자동화.

### 대표 제품
- **Cucumber**  
- **SpecFlow**  
- **Behave**  
- **JBehave**

### 특징
- Given-When-Then 형식  
- 비개발자도 이해하기 쉬운 테스트 문서화  
- QA–개발–기획 협업에 유리

---

# 🧭 **6. 테스트 관리 / 오케스트레이션 도구**
테스트 케이스 관리, 실행, 리포팅을 통합 관리.

### 대표 제품
- **TestRail**  
- **Zephyr**  
- **qTest**  
- **Azure DevOps Test Plans**  
- **Xray (Jira 플러그인)**

### 특징
- 테스트 케이스 관리  
- 실행 기록 및 리포트  
- Jira 등과 연동

---

# 🤖 **7. AI 기반 테스트 자동화 도구**
AI가 테스트 케이스 생성, UI 인식, 유지보수 자동화 등을 지원.

### 대표 제품
- **Testim**  
- **Mabl**  
- **Functionize**  
- **Applitools (시각적 테스트)**

### 특징
- UI 변경에도 자동으로 적응  
- 시각적 비교 기반 테스트  
- 테스트 유지보수 비용 절감

---

# 🧼 **8. 정적 분석 / 코드 품질 검사 도구**
코드를 실행하지 않고 품질과 보안 문제를 분석.

### 대표 제품
- **SonarQube**  
- **ESLint / Pylint / Checkstyle**  
- **Coverity**  
- **Fortify**

### 특징
- 코드 스멜, 보안 취약점 탐지  
- CI 파이프라인에 자동화 가능

---

# 🎯 **정리 요약표**

| 종류 | 목적 | 대표 도구 |
|------|------|-----------|
| 기능(UI) 테스트 | 웹/앱 동작 검증 | Selenium, Cypress, Playwright |
| API 테스트 | 백엔드 기능 검증 | Postman, Rest Assured |
| 성능 테스트 | 부하·응답 시간 측정 | JMeter, Gatling |
| 단위 테스트 | 코드 단위 검증 | JUnit, pytest |
| BDD 테스트 | 시나리오 기반 | Cucumber, SpecFlow |
| 테스트 관리 | 테스트 계획·리포트 | TestRail, Zephyr |
| AI 테스트 | 자동 유지보수·시각적 테스트 | Testim, Mabl |
| 정적 분석 | 코드 품질·보안 | SonarQube, ESLint |

---

- **CI/CD 파이프라인 구성 예시**  
도 만들어줄게.
